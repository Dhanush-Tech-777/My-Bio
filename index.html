Notable Projects
learning Game application for PwDs Student

This project involved designing and developing an Android-based learning game for students with disabilities (PwDs). The goal was to create an interactive and inclusive educational game that helped students improve their skills in areas such as math, language, or cognitive exercises, while also addressing the unique needs of PwDs. The game incorporated several accessibility features such as: Voice Commands & Text-to-Speech (TTS): For visually impaired users, the game used TTS to read out questions and instructions. High-Contrast Mode & Larger Fonts: To support students with low vision, the game included high-contrast themes and customizable font sizes. Simplified Navigation: Touch gestures and larger touch targets were implemented for users with motor impairments, allowing for smoother interaction. Rewards System & Gamification: A points-based system with levels and feedback encouraged continued learning and engagement.

Technologies Used: Android Studio (Java/Kotlin) Firebase (for user progress tracking and data storage) Accessibility Tools in Android (TalkBack, Content Descriptions) Google Cloud Text-to-Speech API Material Design for Android

Challenges & Solutions: Accessibility Design: Ensuring the app was fully accessible required a thorough understanding of different disabilities and how they could impact game interaction. The solution was constant testing and collaboration with users to refine accessibility features. Performance Optimization: The game needed to run smoothly on low-end devices, so optimization techniques like memory management and efficient rendering were implemented. Outcome: The application was well-received during testing, with users noting its ease of use and educational value. The game successfully combined learning and fun, while offering accessibility features that made it usable for a wider audience of students.
